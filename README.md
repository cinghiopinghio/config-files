# Config Files

In this repository I mantain my dotfiles for the following apps:

- vim
- bash
- git
- gnuplot
- screen 
- latexmk
- ...

## Installation:

This repository is meant to be installed with GNU `stow` or `stowsh` (a bash
implementation).

git clone blabla
git submodule init
git submodule update

A Makefile is provided.
